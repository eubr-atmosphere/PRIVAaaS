PRIVAaaS - EUBra-BISEA

Find the sources of PRIVAaaS big data privacy tool at https://github.com/eubr-bigsea/PRIVAaaS.

Find more about EUBra-BISEA at the project website http://www.eubra-bigsea.eu.


You can include the library in your Java application or through its REST API.
The library usage requires two inputs:

  • am anonymization policy, in a json file, (policy example);
  • the data to be anonymized, in one or more json files, (data example).

If you are reading this, you are probably looking to use PRIVAaaS thorough REST.
For this, you should use the HTTP method post to the url: http://localhost:4567/endpoint/.
Tools can help you with that. You can use the postman or the cURL (in your shell) to try it.

Examples are provided with this code. Look for them in PRIVAaaS-client folder.
Another usage example is provided with an html form in: examples/paas-post-form.
